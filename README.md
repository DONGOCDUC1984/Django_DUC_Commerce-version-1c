Giới thiệu:

Django_DUC_Commerce_1c: là trang web bán hàng do ĐỖ NGỌC ĐỨC tự làm toàn bộ.Trang này tương tự như báo "Mua Và Bán".Mọi người đều có thể đăng tin bán hàng.

Công nghệ :

\_Frontend: HTML,CSS, Bootstrap.
